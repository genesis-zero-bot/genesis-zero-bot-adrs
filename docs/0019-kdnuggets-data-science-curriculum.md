# ADR 0019 — KDnuggets Cheat Sheets as Learning Reference

## Status
Proposed

## Context

RegenTribe is building data science competency. Members need structured reference material for technical interview prep. KDnuggets published a curated collection of 100+ cheat sheets across 12 categories.

## Decisions

### D1: Adopt the collection as supplementary reference

Adopt the KDnuggets cheat sheet collection as community learning reference. Integrate it with the existing NVIDIA certification pathway.

### D2: Maintain a curated catalog with quarterly audits

Maintain the collection as a wiki source document. Assign a curriculum owner to audit URLs quarterly.

### D3: Pair with learning tracker

Use the Gherkin learning tracker to track member progress across categories.

## Consequences

### Positive

- Free resource covering all major data science domains.
- Maps to NVIDIA certification curriculum.
- Low maintenance burden.

### Risks

- Some URLs may be stale.
- 100+ sheets can overwhelm new members.

### Mitigations

- Quarterly URL audit by curriculum owner.
- NVIDIA cert roadmap provides structured starting path.
